# c01

A Node / TypeScript project.

Unit tests with Mocha and Chai.

Using the Mongo Driver for database.

---

Install with:
```shell script
npm install
```

Build the program:
```shell script
npm run:build
```

Run the program:
```shell script
npm start
```

Run with auto-reload for development:

```shell script
npm run dev
``` 

Run tests:
```shell script
npm test
```
