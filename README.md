# c01

A Node / TypeScript project.

Unit tests with Mocha and Chai.

Using the Mongo Driver for database.

---

Install with:
```shell
npm install
```

Build the program:
```shell
npm run:build
```

Run the program:
```shell
npm start
```

Run with auto-reload for development:

```shell
npm run dev
``` 

Run tests:
```shell
npm test
```
